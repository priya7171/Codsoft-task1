Title : Brain Tumor Detection And Classification

       I am developing a model in major project name Brain Tumor Detection And Classification by using Deep Learning Algorithm like Deep Convolutional Neural Network and Bidirectional Long-Short Term Memory. Majorly I am using tensorflow and keras library in my model. My role in this project that I am doing coding part of the project. I have taken the dataset of this project from kaggle. In which around 7000 images are there and four classes of Brain Tumor I am are classifying i.e. Meningioma, Gioma, Pituitary and No Tumor. I have used opencv2 library for preprocessing of image. This is short and sweet info about my project.
